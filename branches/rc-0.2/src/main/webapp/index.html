Openhandle

Welcome to the Openhandle web application. More information about Openhandle can be found on the project’s Googlecode homepage.

The form below can be used to submit a query to the Openhandle application. Please specify the Handle ID for which you want to view the record (e.g. 10100/nature).

Optionally, you can also specify:

  1. A comma-separated list of Handle data types to limit your query to
  2. A comma-separated list of integer indexes from the record to return in the response
  3. The format in which the response should be returned
  4. A mimetype to use to return the response as (this is just a convenience feature for viewing responses in a web browser)

(Required)
